# 🧮 Arduino Calculator

Arduino Calculator adalah proyek kalkulator digital sederhana berbasis Arduino yang dirancang untuk melakukan operasi matematika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian. Sistem menerima input dari keypad, memproses data menggunakan mikrokontroler Arduino, lalu menampilkan hasil perhitungan secara langsung pada LCD 16x2.

Proyek ini dibuat sebagai media pembelajaran untuk memahami dasar pemrograman Arduino, penggunaan perangkat input-output, serta penerapan logika matematika dalam sistem embedded.

---

## ✨ Fitur Utama

* ➕ Penjumlahan (Addition)
* ➖ Pengurangan (Subtraction)
* ✖️ Perkalian (Multiplication)
* ➗ Pembagian (Division)
* ⌨️ Input menggunakan Keypad
* 📟 Output ditampilkan pada LCD 16x2
* ⚡ Perhitungan secara real-time
* 🎓 Cocok untuk pembelajaran Arduino dan elektronika dasar

---

## ⚙️ Cara Kerja

1. Arduino melakukan inisialisasi seluruh komponen saat sistem dinyalakan.
2. Pengguna memasukkan angka pertama melalui keypad.
3. Pengguna memilih operator matematika (+, -, ×, ÷).
4. Pengguna memasukkan angka kedua.
5. Saat tombol "=" ditekan, Arduino memproses perhitungan.
6. Hasil perhitungan ditampilkan pada LCD 16x2.
7. Sistem siap menerima perhitungan berikutnya.
